# Cecchinus 📰

**Cecchinus** es un diario online desarrollado con HTML, CSS y JavaScript.
El sitio presenta distintas secciones de noticias como deporte, tecnología, política y espectáculos, con un diseño moderno y responsive.

## 🚀 Características

* Header responsive con menú hamburguesa
* Secciones de noticias organizadas en tarjetas
* Grid layout moderno
* Botón de suscripción destacado
* Footer responsive con las secciones del diario
* Diseño adaptable a dispositivos móviles

## 🛠 Tecnologías utilizadas

* HTML5
* CSS3 (Flexbox y Grid)
* JavaScript
* Bootstrap
